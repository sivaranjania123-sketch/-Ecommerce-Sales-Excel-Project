# 📊 E-commerce Sales Data Analysis (Excel Project)

## 📌 Project Overview
This project analyzes an E-commerce sales dataset using Excel.

## 🛠️ Skills Used
- Data Cleaning
- Excel Formulas (SUM, IF, COUNTIF, XLOOKUP)
- Pivot Tables
- Data Visualization (Bar, Pie, Column Charts)
- Dashboard Creation

## 📂 Dataset
Contains Customers, Products, Stores, and Sales data.

## 📊 Key Insights
- Total Sales: ₹21,70,313
- Female customers contributed more revenue than male customers
- PayPal is the most used payment method
- North region has highest sales
- STORE8 is top performing store

## 📈 Tools Used
- Microsoft Excel

## 📎 Files Included
- Raw Dataset
- Cleaned Data
- Dashboard
- Project Summary

## 🎯 Conclusion
This project demonstrates data cleaning, analysis, and visualization using Excel.
